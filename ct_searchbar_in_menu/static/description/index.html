Search bar in top menu

Get the Search Bar in the menu so customers can search the products from anywhere.

The Search Bar will appear next to the Company Logo.

Configuration

Go to website/settings and simply select the tickbox "Searchbar in menu"

This had to be added as Odoo 12 has built-in multi-websites so you can decide on which websites the searchbar appears.
